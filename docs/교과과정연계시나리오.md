# Pixel OS Horror × 물리 교과과정 연계 시나리오

**작성자:** 원당고등학교 2학년
**연계 교과:** 물리학 — 힘과 에너지 / 전기와 자기 / 빛과 물질
**프로젝트:** Pixel OS Horror (React + Spring Boot 웹 기반 공포 퍼즐 게임)

---

## 1. 연계 개요

### 1.1 연계 방향

Pixel OS Horror의 세계관은 **폐쇄된 연구소 컴퓨터 시스템**이 비정상적으로 재활성화되면서 발생하는 공포 사건을 다룬다. 각 스테이지는 연구소의 서로 다른 구역(층)에 해당하며, 그 구역에서 실제로 수행되던 **물리 실험의 데이터와 장치들**이 퍼즐의 핵심 재료가 된다.

플레이어는 OS 인터페이스 속 메모장(실험 일지), 파일(실험 데이터), 프로그램(실험 장치 제어 소프트웨어)을 조작하며 물리 개념을 직접 적용해 퍼즐을 해결한다.

> **교육적 의도:** 물리 개념을 '지식 암기'가 아닌 '퍼즐 해결 도구'로 경험하게 함으로써 능동적 학습을 유도한다.

### 1.2 스테이지별 연계 교과 단원 요약

| 스테이지 | 공간 배경 | 연계 단원 | 핵심 개념 |
|---|---|---|---|
| Stage 1 | 전기 실험실 (1층) | 전기와 자기 — 전기 | 전기장, 전위차, 저항 연결, 전류의 자기 작용 |
| Stage 2 | 광학 실험실 (2층) | 빛과 물질 — 빛의 이해 | 빛의 간섭, 굴절, 빛과 물질의 이중성 |
| Stage 3 | 지하 에너지 격납고 | 힘과 에너지 + 빛과 물질 — 물질의 이해 | 역학적 에너지 보존, 에너지띠, 특수 상대성 이론 |

## 2. 세계관 및 전체 내러티브

### 배경 설정

> 2031년. 국립 물리 연구소 'CENTRAL-K'가 원인불명의 정전으로 봉쇄된 지 3년이 지났다.
> 오늘 밤, 연구소 내부 서버가 외부 신호 없이 스스로 재부팅되었다.
> 당신은 원격으로 연구소 내부 OS에 접속한다.
> 화면에 나타나는 것은 낯선 바탕화면과, 켜지면 안 되는 실험 장치들이다.

각 스테이지의 OS는 서로 다른 실험실의 로컬 컴퓨터다. 플레이어는 **메모장에 남겨진 연구원의 실험 일지**를 단서 삼아, **파일 속 실험 데이터를 해독**하고, **프로그램(실험 장치 제어 SW)을 작동**시켜 해당 구역의 봉쇄를 해제하며 진행한다.

## 3. Stage 1 — 전기 실험실 (1층)

### 3.1 연계 단원

| 교과서 단원 | 페이지 | 연계 내용 |
|---|---|---|
| 전기장과 전위차 | p.72 | 전구 ON/OFF = 전위 높낮이(전위차 개념) |
| 저항의 연결과 소비 전력 | p.76 | 꼬인 전선 추적 = 직렬/병렬 회로 탐색 |
| 전류의 자기 작용 | p.98 | 전류 흐름 방향 = 자기장 방향 퍼즐 힌트 |

### 3.2 스토리 시나리오

```
[메모장 — 실험 일지 #3 (연구원 K 작성)]

23.11.14
오늘 전기 회로 실험을 마쳤다.
4개 조명 회로를 점검하는데 배선이 꼬여서
어느 스위치가 어느 등에 연결됐는지 알기 어렵다.

전위차가 있어야 전류가 흐른다.
켜진 등(높은 전위) → 1
꺼진 등(낮은 전위) → 0

배선을 따라가면 반드시 답을 찾을 수 있다.
```

### 3.3 퍼즐 설계 (현재 구현 반영)

**전기 회로 퍼즐 (`CircuitPuzzleWindow`)**

- 전구 4개가 랜덤으로 ON/OFF 상태로 제시됨
- 전선이 ㄷ/역ㄷ 형태로 교차하여 어느 박스가 어느 전구와 연결됐는지 추적해야 함
- 각 박스의 값을 클릭해 0↔1 토글
- **전구 ON → 연결된 박스 = 1 / 전구 OFF → 연결된 박스 = 0** 조건 달성 시 통과

**물리 개념 적용 포인트:**
- 전위차(ΔV): 전구 켜짐 = 높은 전위, 꺼짐 = 낮은 전위 → 박스 값 1과 0으로 표현
- 전류의 방향성: 꼬인 전선 추적 = 회로에서 전류 경로 탐색
- 소비 전력: 메모장에 각 전구의 저항 값이 주어지고, 특정 저항 조합을 맞추는 심화 힌트 가능

### 3.4 공포 연출 연계


> **centralkeeper** 실행 순간 — 회로 전류가 역류하는 연출
> 화면이 짧게 백색으로 플래시되며 "전위차 역전 감지 — 시스템 불안정" 텍스트 출력
> 배경에서 전기 방전 소리 재생 (Web Audio API)

## 4. Stage 2 — 광학 실험실 (2층)

### 4.1 연계 단원

| 교과서 단원 | 페이지 | 연계 내용 |
|---|---|---|
| 빛의 중첩과 간섭 | p.124 | 이중 슬릿 간섭 패턴 → 암호 도출 |
| 빛의 굴절 | p.130 | 굴절각 계산 → 레이저 경로 퍼즐 |
| 빛과 물질의 이중성 | p.136 | 파동/입자 이중성 힌트 → 두 파일 모두 해결해야 정답 |

### 4.2 스토리 시나리오

```
[메모장 — 기록1.txt (연구원 M 작성)]

이중 슬릿 실험 결과 기록.
스크린에 나타난 간섭무늬의 밝은 띠 위치:

  d·sinθ = mλ  (m = 0, ±1, ±2 ...)

밝은 무늬 순서가 오늘의 점검 코드다.
무늬 번호를 순서대로 입력하라.
```

```
[메모장 — 기록2.txt (연구원 M 작성)]

레이저 굴절 실험.
물질 A에서 물질 B로 입사각 30°.
스넬의 법칙:  n₁sinθ₁ = n₂sinθ₂

n₁ = 1.0, n₂ = 1.5 일 때
굴절각 θ₂의 sin 값을 분수로 표현하면?

분자 _ 분모 _ → 이것이 키워드다.
```

### 4.3 퍼즐 설계

**파일 1 — 기호 패턴 퍼즐 (암호1)**
- 이중 슬릿 간섭 무늬 이미지(SVG) 제시
- 밝은 무늬(보강 간섭)와 어두운 무늬(상쇄 간섭) 패턴에서 숫자 코드 추출
- 입력: 4자리 숫자 코드 (예: `0121`)

**파일 2 — 단어 조합 퍼즐 (암호2)**
- 스넬의 법칙 계산 결과에서 단어 도출
- sin30° = 0.5, n₂sinθ₂ = 1.0 × 0.5 = 0.5, θ₂ = sin⁻¹(0.5/1.5) = sin⁻¹(1/3)
- "1/3" → 힌트 메모장의 테이블에서 해당 위치의 단어 = `SHADOW`

**물리 개념 적용 포인트:**
- 보강 간섭 조건(mλ = dsinθ)을 직접 계산해야 패턴 순서를 알 수 있음
- 스넬의 법칙을 계산하지 않으면 단어 키워드 도출 불가
- **빛의 이중성 연결:** 파일 2개를 모두 해결해야 프로그램이 열림 → "파동이면서 동시에 입자" = 두 가지 성질을 모두 갖춰야 한다는 메타포

### 4.4 공포 연출 연계

> 프로그램 실행 시 광학 실험 장치가 "오작동"하는 연출
> 화면의 아이콘들이 빛의 회절처럼 퍼지고 겹치는 CSS 애니메이션
> "간섭 패턴 붕괴 — 위상 역전" 텍스트와 함께 창 닫기 버튼 비활성화

## 5. Stage 3 — 지하 에너지 격납고

### 5.1 연계 단원

| 교과서 단원 | 페이지 | 연계 내용 |
|---|---|---|
| 역학적 에너지 보존 | p.42 | 에너지 총량 보존 = 퍼즐 잠금 해제 조건 |
| 열효율 | p.52 | 시스템 효율 계산 → 암호 도출 |
| 에너지띠와 반도체 | — | 도체/부도체/반도체 분류 → 회로 연결 방식 결정 |
| 특수 상대성 이론 | — | 시간 왜곡 서사 + 공포 연출의 핵심 트리거 |

### 5.2 스토리 시나리오

```
[메모장 — FINAL_LOG.txt (작성자 불명)]

에너지는 보존된다.
Ek + Ep = 일정

격납고 에너지 레벨이 임계치를 초과했다.
총 역학적 에너지: 500 J
현재 운동 에너지: 200 J
위치 에너지는 얼마인가?

그 숫자가 격납고 잠금 해제 코드다.

-- 경고 --
이 시스템에 접속한 시간이 너무 길어지면
상대성 이론에 의해 당신의 시간은 왜곡된다.
빠져나가라.
```

### 5.3 퍼즐 설계

**파일 1 — 역학적 에너지 보존 퍼즐**
- 메모장에 `Ek = ½mv²`, `Ep = mgh` 공식과 수치 제공
- Ek + Ep = 500 J, Ek = 200 J → Ep = **300 J** → 코드: `300`
- 힌트 2: 열효율 η = W/Q × 100 계산 → 추가 자릿수 도출 (예: η = 60% → 코드 `300-60`)

**파일 2 (숨김 파일) — 에너지띠 반도체 퍼즐**
- 도체/반도체/부도체 분류 문제
- 4가지 물질 카드를 올바른 에너지띠 그림에 드래그하면 잠금 해제
- 실제 구현: 4개 버튼 클릭으로 도체(1) / 부도체(0) 분류 → Stage 1 전기 회로 퍼즐과 구조적 연결

**프로그램 — 특수 상대성 이론 연출**
- 프로그램 실행 시 카운트다운 타이머 표시: "시스템 접속 경과 시간"
- 타이머가 빠르게 흐르다 점점 느려지는 애니메이션 (시간 지연 시각화)
- 배경음: 저주파 진동음 + "로런츠 인자 γ > 1.0 감지" 텍스트
- **최종 공포 연출:** 화면의 모든 텍스트가 뒤집히고 시간이 역행하는 듯한 이미지

### 5.4 특수 상대성 이론 연계 서사

> 시스템이 재활성화된 이유는 격납고 내부에서 고에너지 입자가 **광속에 가까운 속도로 순환**하기 때문이었다.
> 로런츠 인자 γ에 의해 입자 입장에서의 시간은 거의 흐르지 않았지만,
> 시스템 내부의 3년은 입자에게는 단 몇 초였다.
> 당신이 접속한 이 순간, 시스템은 그 입자의 시간을 기준으로 작동하고 있다.

## 7. 학습 효과 및 평가 포인트

### 7.1 기대 학습 효과

1. **맥락적 이해:** 공식을 외우는 것이 아니라 퍼즐 해결에 공식을 적용함으로써 개념의 실용성을 체험
2. **자기주도 탐구:** 힌트(메모장)만 보고 플레이어가 스스로 계산하여 정답 도출
3. **단원 간 연결:** Stage 1(전기) → Stage 2(빛) → Stage 3(에너지+상대성) 순서가 교과서 단원 흐름과 일치

### 7.2 발표·평가 시 강조 포인트

- **Stage 1 전기 회로 퍼즐:** "전위차 개념을 0과 1로 디지털화하여 게임 메카닉으로 구현"
- **Stage 2 간섭 퍼즐:** "보강/상쇄 간섭 조건(mλ = dsinθ)을 직접 계산해야 진행 가능"
- **Stage 3 에너지 보존 퍼즐:** "역학적 에너지 보존 법칙을 암호 체계로 치환"
- **특수 상대성 이론 연출:** "물리 개념(시간 지연)을 공포 서사의 핵심 동인으로 활용"
